1. header;
2. content:
* description - "Главные проблемы в обучении";
* techniques - "Техники обучения";
* video - "Видео на TED";
* oakley - "История Барбары Оакли";
* feynman - "Метод Фейнмана";
* digits - "Цифры и факты";

Применена технология выравнивания и построения сетки.
